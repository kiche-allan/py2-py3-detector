1. The Basic Concept: What is MRO?
In multiple inheritance, if Class C inherits from A and B, and both A and B have a method called save(), which one does C use?

Python 2 (Old): Uses DFS (Depth-First Search). It goes as deep as possible into the first parent before looking at the second.

Python 3 (New): Uses C3 Linearization. It ensures that a child always comes before its parents and that the order of parents is preserved, avoiding the "Diamond Problem."

DIAMOND INHERITANCE
class A:
    def method(self): return "A"

class B(A):
    pass

class C(A):
    def method(self): return "C"

class D(B, C):
    pass

D().method()
# Python 2: returns "A"  ← goes deep into B first, finds A
# Python 3: returns "C"  ← C3 algorithm finds C first


Scenario 2: super() syntax change
python# Python 2
class B(A):
    def method(self):
        super(B, self).method()   # must pass class and self explicitly

# Python 3
class B(A):
    def method(self):
        super().method()           # no arguments needed
Python 2 code using super() without arguments breaks in Python 2. Python 3 code using super() without arguments does not exist in Python 2. Both are detectable with regex.
